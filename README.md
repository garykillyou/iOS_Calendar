# iOS_Calendar

多人專案

一個簡單的行事曆，可以在日期上儲存事情。

使用Xcode搭配Objective-C撰寫，都是Apple本身的API，雖然有相容C++的語法，但是Objective-C的operator與C++和Java大相逕庭，一開始也是認得挺辛苦的。
因為是iOS是封閉式系統，如果沒有Mac會寫得非常火大。
